###### 1. git 명령어 정리  

특정 파일만 올리기

git add <파일명>

변경된 모든 파일 staging 올리기

git add *


git commit -m "commit message"



현재 변경된 파일 상태 보기
git status

변경된 부분 비교
git diff 파일명

원격 저장소에 올리기
git push origin[저장소alias] master[branch명]
